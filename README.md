# ozon-price-calculator
Ozon Price Automation Tool — это приложение на Python/Streamlit для продавцов на Ozon. Оно решает две главные задачи:  Автоматически находит себестоимость товаров в прайс-листах поставщиков (умеет распознавать штучные товары и наборы).  Рассчитывает итоговую розничную цену для загрузки на Ozon, учитывая комиссию, логистику и желаемую маржу.
